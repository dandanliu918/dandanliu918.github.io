Interview questions

1 rem和em的区别？

  • rem 单位翻译为像素是由html元素字体大小决定
  • em 单位转为像素值 取决于父元素使用的字体大小

2 什么是作用域？

  • 函数执行时候会形成一个执行环境，这个环境叫做作用域

3 jq中有一个方法只执行一次 然后第二次不起作用

  • $.one()

4 bind和live的区别？

如果是通过jq添加的层和对象一定要用live 其他不起作用

live 的确定就是运行完空间不释放，太多的使用会占用更多的内存，bind则点击以后就释放空间

  • JQuery从1.7版本开始将bind(),live()和delegate()方法合并成了on()方法了
  • 同样unbind(),die()和undelegate()方法也合并成了off()方法

5 box-sizing 属性

  1. content-box(default) :border和padding 不计算width之内
  2. border-box ：border和padding计算width之内

  3. padding-box：padding计算width之内

    注意：IE8+浏览器支持content-box和border-box 火狐则支持全部三个值

AJAX

原理

  • 异步的JavaScript和xml，是一种创建交互式网页应用的网页开发技术

优点

  • 1 无刷新更新数据
  • 2 异步与服务器通信
  • 3 前端和后端负载平衡
  • 4 基于标准被广泛支持
  • 5 界面与应用分离

缺点

  • 1 Ajax干掉了back和history功能，即对浏览器机制的破坏
  • 2 Ajax安全问题
  • 3 对搜索引擎支持较弱
  • 4 破坏程序的异常处理机制
  • 5 违背url和资源定位的初衷
  • 6 Ajax不能很好支持移动设备
  • 7 客户端过肥，太多客户端代码造成开发成本升高

Ajax引擎（XMLHttpRequest）

  • Ajax引擎就是XMLHttpRequest对象，所有现代浏览器均支持XMLHttpRequest对象（IE5和IE6使用的ActiveXObeject）

    适用于IE7之前的版本

    function createXHR() {

        if(typeof  arguments.callee.activeXString != "string"){
        var versions= ["MSXML2.XMLHttp.6.0","MSXML2.XMLHttp.3.0","MSXML2.XMLHttp"];
        for(var i=0;i<versions.length;i++){
          try{
              new ActiveXObject(versions[i]);
              arguments.callee.activeXString=versions[i];
              break;
          }catch(e){

          }
        }
    }
    return new new ActiveXObject(arguments.callee.activeXString);
}

Ajax引擎的状态属性readyState（0 1 2 3 4）

  • 0 请求初始化 var xml=new XMLHttpRequest();
  • 1 服务器链接已经建立 xml.open()
  • 2 请求已经发送
  • 3 请求已经接收
  • 4 请求已经完成 xml.send()

CSS3-过渡

CSS3 过渡属性

transition

  • transition-property 指定过渡 或者动态模拟的css属性
none 默认值 没有
all 默认值 表示指定元素都支持transition-property 属性
single-transition-property 指定样式 一下可以过渡的样式
 - 颜色属性 红 绿 蓝 和透明度组合的过渡属性， background-color border-color color outline-color
 - 具有长度值 百分比值属性 真实数字 word-spacing width vertical top 
 - integer 离散不足 outline-offset z-index
 - number zoom opacity font-weight
 - 变形系列属性  rotate rotate3d scale skew translate ...
 - retangle 通过x y width height 转为数值 crop 属性
 - visibilty 离散步骤 0-1 范围内 0 表示隐藏 1 表示全部显示
 - 阴影 color x y blur text-shadow
 - 渐变（gradient） 通过每次停止时的位置和颜色进行变化
 - paint server（SVG） 只支持 gradient-gradient color-color
 - space-separated list of above 如果列表中有项目值 就按照上面的规则进行变化 否则无变化
 - 缩写属性 参照书本table（page-422）
  • transition-duration 完成过渡所需要的时间
- time为数值 s ms
  • transition-timing-function 过渡的函数
- ease 默认
- linear 元素样式从初始状态到终止状态速度是恒速
- ease-in 渐显效果
- ease-out 渐隐效果
- ease-in-out 渐显渐隐效果
  • transition-delay 过渡开始出现的延迟时间
time 
- 正整数 元素过渡定制不会立即触发 当过了设定时间值之后才会触发
- 负整数 元素过渡动作从该时间点开始显示 之前的动作被截断
- 0 没有任何触发

CSS3 触发过渡

  • 伪类元素触发
1 ：active 表示用户点击某个元素并按住鼠标按钮显示的状态
2 ：focus 通常会在元素接收键盘焦点时出现
3 ：checked 发生在input  ....

  • 媒体触发

    @media only screen and （max-width:960px）{….}

CSS-Border

边框的基本属性

border-width 设置元素边框的粗细
border-color 设置元素边框的颜色
border-style 设置元素边框的类型

- 简写：border:width color solid

边框的类型

  • border-style

    属性：
  none 定义无边框
  hidden 隐藏 
  dotted 点状边框
  deshed 虚线边框
  solid 实现边框
  double 双线边框
  groove 3D凹槽边框
  ridge 3D垄状边框
  inset 3Dinset边框
  outset 3Doutset边框
  inherit 务样式 继承父类样式 （部分浏览器不支持）

  • border-image 低版本不支持

    属性：
none 无定义
image 设置边框背景
number 数值 边框的大小 1-4 这四个值表示方位
percentage 设置边框的大小 百分比
stretch 拉伸
repeat 重复
round 平铺边框背景

剖析：
border-images-source 引入图片背景
border-images-slice 切割引入图片背景
border-images-width 边框图片的宽度

  • border-radius 9+

    属性：
none 无
length 值1 四个圆角的效果 1-4值分别代表角度
length 值2 两个值 左上角和右下角取第一样值，右上角和左下角取第二个值
length 值3 设置三个值 第一个top-left 第二个 top-right和bottom-left 第三个bottom-right
length 值4 四个圆角分别取不同的值 第一个设置top-left 第二个 top-right 第三个bottom-right 最后一个bottom-left
  -注意： 这里length 不支持负值

例子：border-radius:1px 2px 3px 3px / 1px 2px 3px 3px
                   水平方向           垂直方向

  • border-shadow 9+

    属性：
none 无
inset 内嵌阴影 
x-offset 阴影水平偏移 正值->右边 负值->左边 
y-offset 阴影垂直偏移 正值->底部 负值->顶部
blur-radius 模糊半径 0 不具有 值越大模糊越大
spread-radius 扩展半径 正值->扩大 负值->缩小
color 颜色

低版本：
filter:progid:DXImagetransform.Micsoft.Shadow(color="#ccc",Direction=20(阴影角度),Strength=30(阴影半径))；
DropShadow 盒状阴影

CSS3-渐变

线性渐变的基本属性

  • linear-gradien 线性渐变
  angle 通过角度来确定渐变方向
  linear-gradien：<point>||<angle>?stop,stop，stop

注意：浏览器 webkit(-webkit-) geeko(-moz-) presto(-o-) Trident(-ms-)

 to top 从顶部到底部渐变 0deg 360deg -360deg
 to bottom 从底部到顶部渐变 180deg -180deg
 to left 从右向左渐变 90deg -270deg
 to right 从左到右渐变 -90deg 270deg
 to top left 从右下角向左上角线性渐变
 to bottom left 从右上角到左下角线性渐变
 to bottom right 从左上角到右下角线性渐变
  • radial-gradient
 - position 用来设置定义径向渐变的圆心位置

        length  用长度指定径向渐变的圆心的横坐标或者纵坐标
        percentage 用百分比指定径向渐变圆心的横坐标或者纵坐标 （支持负值）
        left 设置左边为径向渐变圆心的横坐标值
        right 设置右边为径向渐变圆心的横坐标值
        top 设置顶部为径向渐变圆心的纵坐标值
        bottom 设置底部为径向渐变圆心的纵坐标值

 - shape 用来定义径向渐变的形状

       circle 用来指定圆形的径向渐变
       ellipse 用来指定椭圆形的径向渐变

 - size 用来确定径向渐变的的结束的形状大小

        closest-side 指定径向渐变的半径长度从圆心到离圆心最近的边
        closest-corner 指定径向渐变的半径长度从圆心到离圆心最近的角
        farthest-side 指定径向渐变的半径长度从圆心到离圆心最远的边
        farthest-corner 指定径向渐变的半径长度从圆心到离圆心最远的角

- color-stop 径向渐变线上的停止颜色

CSS3径向渐变的基本用法

- 圆心相同
- 椭圆形状径向渐变
- 渐变形状配合圆心定位
  - center = at 50% 50%
  - top = at 50% 0%
  - right = at 100% 50%
  - bottom = at 50% 100%
  - left = at 0% 50% 
  - top left = at 0 0
  - top right = at 100% 0%
  - bottom right = at 100% 100%
