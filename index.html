CSS3-过渡

CSS3 过渡属性

transition

  • transition-property 指定过渡 或者动态模拟的css属性
none 默认值 没有
all 默认值 表示指定元素都支持transition-property 属性
single-transition-property 指定样式 一下可以过渡的样式
 - 颜色属性 红 绿 蓝 和透明度组合的过渡属性， background-color border-color color outline-color
 - 具有长度值 百分比值属性 真实数字 word-spacing width vertical top 
 - integer 离散不足 outline-offset z-index
 - number zoom opacity font-weight
 - 变形系列属性  rotate rotate3d scale skew translate ...
 - retangle 通过x y width height 转为数值 crop 属性
 - visibilty 离散步骤 0-1 范围内 0 表示隐藏 1 表示全部显示
 - 阴影 color x y blur text-shadow
 - 渐变（gradient） 通过每次停止时的位置和颜色进行变化
 - paint server（SVG） 只支持 gradient-gradient color-color
 - space-separated list of above 如果列表中有项目值 就按照上面的规则进行变化 否则无变化
 - 缩写属性 参照书本table（page-422）
  • transition-duration 完成过渡所需要的时间
- time为数值 s ms
  • transition-timing-function 过渡的函数
- ease 默认
- linear 元素样式从初始状态到终止状态速度是恒速
- ease-in 渐显效果
- ease-out 渐隐效果
- ease-in-out 渐显渐隐效果
  • transition-delay 过渡开始出现的延迟时间
time 
- 正整数 元素过渡定制不会立即触发 当过了设定时间值之后才会触发
- 负整数 元素过渡动作从该时间点开始显示 之前的动作被截断
- 0 没有任何触发

CSS3 触发过渡

  • 伪类元素触发

 - position 用来设置定义径向渐变的圆心位置

        length  用长度指定径向渐变的圆心的横坐标或者纵坐标
        percentage 用百分比指定径向渐变圆心的横坐标或者纵坐标 （支持负值）
        left 设置左边为径向渐变圆心的横坐标值
        right 设置右边为径向渐变圆心的横坐标值
        top 设置顶部为径向渐变圆心的纵坐标值
        bottom 设置底部为径向渐变圆心的纵坐标值

 - shape 用来定义径向渐变的形状

       circle 用来指定圆形的径向渐变
       ellipse 用来指定椭圆形的径向渐变

 - size 用来确定径向渐变的的结束的形状大小

        closest-side 指定径向渐变的半径长度从圆心到离圆心最近的边
        closest-corner 指定径向渐变的半径长度从圆心到离圆心最近的角
        farthest-side 指定径向渐变的半径长度从圆心到离圆心最远的边
        farthest-corner 指定径向渐变的半径长度从圆心到离圆心最远的角

- color-stop 径向渐变线上的停止颜色

CSS3径向渐变的基本用法

- 圆心相同
- 椭圆形状径向渐变
- 渐变形状配合圆心定位
  - center = at 50% 50%
  - top = at 50% 0%
  - right = at 100% 50%
  - bottom = at 50% 100%
  - left = at 0% 50% 
  - top left = at 0 0
  - top right = at 100% 0%
  - bottom right = at 100% 100% 
  - bottom left = at 0% 100%

CSS3-盒模型 flexbox

flexbox 模型的基础知识

  • display 伸缩容器设置

    box 设置为块伸缩容器
inline-box 设置为内联级伸缩容器

  • box-orient 伸缩流的方向

    horizontal 从左到右在一条平行线上显示
veritical 从上到下在一条垂直线上显示
inline-axis 沿着内联轴排列显示
block-axis 沿着块轴排列显示

  • box-direction 布局顺序

    normal 正常顺序
reverse 倒序

  • box-lines 伸缩换行

    single 伸缩容器多行或多列显示伸缩项目
multiple 指定容器多行或者多列显示伸缩项目，当伸缩容器没有足够空间 就会自动换行或多列显示

  • box-pack 主轴对齐

    start 伸缩项目向一行的起始位置靠齐
end  和start相反
center 伸缩项目向中间靠齐
justify 伸缩项目会平均分布在一行里

  • box-align 侧轴对齐

    start 伸缩容器项目顶部放置在伸缩容器的顶端，伸缩容器的额外空间放置在伸缩项目低端
end 与start 相反
center 伸缩项目紧靠一起，垂直居中于伸缩容器
baseline 伸缩项目根据他们基线对齐
stretch 伸缩项目填充整个伸缩容器

  • box-flex 伸缩性

    number 一般是默认三个 如果一个伸缩项目设置为2 另一个默认为1

  • box-ordinal-group 显示顺序

    integer 默认值为1 为正整数 主要设置伸缩项目在伸缩容器的位置顺序列号 值越大 伸缩项目月排在后面

混合版flexbox 模型的基本使用

  • display 伸缩容器设置

    flexbox 设置元素为块级伸缩容器
inline-flexbox 设置元素为内联块伸缩容器

  • flex-direction 伸缩流方向

    row 初始值 =ltr 从左到右 =rtl 从右向左
row-reverse  与row 方向相反
column 伸缩项目按着文档流出现的顺序从上到下排列
column-reverse 伸缩项目按着文档流出现的顺序从下到上排列

    • flex-wrap 伸缩换行

      nowrap 默认值
      wrap 伸缩容器的所有伸缩项目在伸缩项目时，伸缩项目就会移除容器 自动换行 以多行多列显示
      wrap-reverse 与wrap 相似 但是与wrap方向相反

    • flex-flow 伸缩方向与换行

      flex-direction 用来设置伸缩容器的伸缩流方向
flex-wrap 用来设置伸缩容器中伸缩项目在伸缩容器无足够空间的时候伸缩项目在伸缩容器中是否换行排列

    • flex-pack 主轴对齐

      start 伸缩项目向一行的起始位置靠齐
end  和start相反
center 伸缩项目向中间靠齐
justify 伸缩项目会平均分布在一行里
distribute 伸缩项目会平均分布在一行里 两边会保留一半的空间

    • flex-align 侧轴对齐

      start 伸缩容器项目顶部放置在伸缩容器的顶端，伸缩容器的额外空间放置在伸缩项目低端
end 与start 相反
center 伸缩项目紧靠一起，垂直居中于伸缩容器
baseline 伸缩项目根据他们基线对齐
stretch 伸缩项目填充整个伸缩容器

    • flex-line-pack 堆栈伸缩行

      start 伸缩项目行想布局轴（伸缩容器的侧轴）的起始位置靠齐
end 伸缩项目行向布局轴（伸缩容器的侧轴）的结尾位置靠齐
center .......中间位置靠齐
justify 平均分布 第一个在顶部 最后一个在底部
distribute 伸缩项目平均分布在布局轴 两端保留一半空间
stretch 伸缩项目拉伸填充整个伸缩容器

    • flex 伸缩性

      positive-flex 设置正弹性的整数 如果忽略 该项目的正弹性为1 负值无效
negaive-flex 设置负弹性的整数 如果忽略 该项目的正弹性为0 负值无效
preferred-size 伸缩项目的首选大小
none 如果忽略 首选大小默认值为0

    • flex-order 显示顺序

      integer 是一个自然数 默认值为0 用来设置伸缩项目的位置顺序号

新版本flexbox模型的基本使用

  • display 伸缩容器

    flex  设置块伸缩容器
inline-flex 设置内联伸缩容器

    row ltr排版=从左向右排 rtl排版=从右向左排列

  • flex-wrap 伸缩换行

    nowrap 伸缩容器单行显示 ltr= 从左到右 rtl=从右到左

wrap-reverse 与wrap相反

    -flex-flow 伸缩流方向与换行

  • align-items 和align-self 侧轴对齐

  • flex 伸缩性

flex-basis length |auto 设置伸缩项目的基准值
