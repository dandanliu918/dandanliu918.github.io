CSS-Animation

CSS3动画属性

animation

  • animation-name 关键帧名字 @keyframes
方法一
 @keyframes name{
   form{...}
   percentage{...}
   to{...}
 }

方法二
@keyframes name{
      0%{...}
      50%{...}
      100%{...}
  }
  • animation-duration 设置动画播放所需要时间 秒为单位
  • animation-timing-function 设置动画播放方式
  • animation-delay 设置动画播放开始时间 一般以秒为单位
  • animation-iteration-count 设置动画播放循环次数 infinite
  • animation-direction 设置动画播放方向 alternate 偶次向前播放
  • animation-paly-state 设置动画播放状态 runing paused
  • animation-fill-mode 设置动画时间外的属性 none forwards backwards both
