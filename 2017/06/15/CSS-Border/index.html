CSS-Border

边框的基本属性

border-width 设置元素边框的粗细
border-color 设置元素边框的颜色
border-style 设置元素边框的类型

- 简写：border:width color solid

边框的类型

  • border-style

    属性：
  none 定义无边框
  hidden 隐藏 
  dotted 点状边框
  deshed 虚线边框
  solid 实现边框
  double 双线边框
  groove 3D凹槽边框
  ridge 3D垄状边框
  inset 3Dinset边框
  outset 3Doutset边框
  inherit 务样式 继承父类样式 （部分浏览器不支持）

  • border-image 低版本不支持

    属性：
none 无定义
image 设置边框背景
number 数值 边框的大小 1-4 这四个值表示方位
percentage 设置边框的大小 百分比
stretch 拉伸
repeat 重复
round 平铺边框背景

剖析：
border-images-source 引入图片背景
border-images-slice 切割引入图片背景
border-images-width 边框图片的宽度

  • border-radius 9+

    属性：
none 无
length 值1 四个圆角的效果 1-4值分别代表角度
length 值2 两个值 左上角和右下角取第一样值，右上角和左下角取第二个值
length 值3 设置三个值 第一个top-left 第二个 top-right和bottom-left 第三个bottom-right
length 值4 四个圆角分别取不同的值 第一个设置top-left 第二个 top-right 第三个bottom-right 最后一个bottom-left
  -注意： 这里length 不支持负值

例子：border-radius:1px 2px 3px 3px / 1px 2px 3px 3px
                   水平方向           垂直方向

  • border-shadow 9+

    属性：
none 无
inset 内嵌阴影 
x-offset 阴影水平偏移 正值->右边 负值->左边 
y-offset 阴影垂直偏移 正值->底部 负值->顶部
blur-radius 模糊半径 0 不具有 值越大模糊越大
spread-radius 扩展半径 正值->扩大 负值->缩小
color 颜色

低版本：
filter:progid:DXImagetransform.Micsoft.Shadow(color="#ccc",Direction=20(阴影角度),Strength=30(阴影半径))；
DropShadow 盒状阴影
