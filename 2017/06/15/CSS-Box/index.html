CSS3-盒子

CSS 盒子模型

  • W3C标准

    外盒尺寸计算（元素空间尺寸）
element 空间高度 = 内容高度+内距+边框+外距
element 空间宽度 = 内容宽度+内距+边框+外距
内盒子尺寸计算（元素大小）
element 空间高度 = 内容高度+内距+边框（height 为内容高度）
element 空间宽度 = 内容宽度+内距+边框（width 为内容宽度）

  • IE6以下 

    外盒尺寸计算（元素空间尺寸）
element 空间高度 = 内容高度+外距（height 包含了 内容高度 边框 内距）
element 空间宽度 = 内容宽度+外距（width 包含了 内容宽度 边框 内距）
内盒子尺寸计算（元素大小）   
element 空间高度 = 内容高度（height 包含了 内容高度 边框 内距）
element 空间宽度 = 内容宽度（height 包含了 内容宽度 边框 内距）

CSS盒子基本属性

  • box-sizing 8+

    content-box 默认值
border-box 包含边框的内容
inherit 继承父类样式

  • overflow-x和overflow-y 内容溢出 水平和垂直

    visible 默认值
auto 自动
hidden 隐藏
scroll 滚动条
no-display 当内容溢出容器时不显示元素
no-content 当内容溢出容器时不显示内容

  • resize 自由缩放属性 改变元素尺寸大小

    none
both 用户不能拖动元素 同时修改元素的宽度和高度
horizontal 用户可以拖动元素 可以修改宽度 高度不可以
vertical 用户可以拖动宽度 不能修改高度
inherit 继承父类元素属性值

    • outline 外轮廓属性 8+

      outline-color 轮廓线的颜色
      outline-style 轮廓线的样式
      outline-width 轮廓线的宽度
      outline-offset 轮廓线偏移位置数值（可以是负值）
      inherit 继承父元素的outline效果
