CSS3-选择器

CSS选择器

!important 强调作用
* 全局
el 标签元素
# id
class  类
selected，selected 群组选择器
el f 后代选择器 7+
el >f 子选择器 7+
el+f 相邻兄弟 7+
el~f 通用选择器 7+
e:link 链接伪类选择器
e:visited 链接伪类选择器
e:active 用户行为伪类选择器 8+
e:hover 用户行为伪类选择器 
e:focus 用户行为伪类选择器 8+
e:target 目标伪类选择器 8+
 - target 例子 高亮区块显示 tabs效果 灯箱效果 相册效果
e:lang 语言类选择器  8+

UI元素状态伪类选择器

e:checked 选中状态伪类选择器 9+
e:enabled 启用状态伪类选择器 9+
e:disabled 不可用状态伪类选择器 9+

结构伪类选择器

e:first-child 第一个子元素 9+
e:last-child 最后一个子元素  9+
e:root 匹配元素e的根元素  9+
e f:nth-child(n) 选择e 下的f 第n个元素  9+
e f:nth-last-child(n) 选择e 下的f 第最后n个元素  9+
e:nth-of-type(n) 选择父元素内指定的第n个e元素  9+
e:nth-last-of-type(n) 选择父元素内指定的倒数n个e元素  9+
e:first-of-type 选择父元素指定的第一个元素  9+
e:last-of-type 选择父元素指定的第一个元素  9+
e:only-of-type 选择父元素只包含一个元素 子元素匹配e元素  9+
e:only-type 选择父元素只包含一个元素 子元素匹配e元素  9+
e:empty 选择没有子元素而且该元素也不包含任何文本节点  9+
  - 注意： n的值可以为 odd 奇数 even 偶数 或者 数 /2n+1 ....根据需求

否定伪类选择器

e:not(f)匹配所除f之外 9+

伪元素

e::first-letter 匹配选择文本第一个字母
e::first-line 匹配选择文本第一个行
e::before ...之前
e::after  ...之后
e::selected 默认突出文本效果 只接受两个属性 background color

属性选择器语法

e[attr] 选择匹配具有attr的e元素 7+
e[attr=val] 选择匹配具有attr的e元素 并且attr属性值val 7+
e[attr|=val] 选择匹配具有attr的e元素 并且attr属性值val 或者以val开始的值 7+
e[attr~=val] 选择匹配具有attr的e元素 并且attr具有多个空格分割值  其中一个值就是val 7+
e[attr*=val] 选择匹配具有attr的e元素 并且attr=val 全部 7+
e[attr^=val] 选择匹配具有attr的e元素 并且attr=val 开头 7+
e[attr$=val] 选择匹配具有attr的e元素 并且attr=val 结尾 7+
