CSS-Background

背景的基本属性

background-color 背景颜色
background-image 背景图片
background-repeat 背景图片展开方式
background-attachment  背景图片是固定和是滚动
background-position  背景图片位置
background-origin  背景图片的起点
background-clip  背景图片的显示范围
background-size  背景图片的尺寸大小
background-break  指定内联元素背景图片进行平铺时的循环方式

  • background-repeat 

    属性:
repeat 平铺
repeat-x x轴平铺
repeat-y y轴平铺
no-repeat 不平铺

  • background-attachment 

    属性：
scroll 滚动
fixed 固定不动

  • background-position 

    属性：此处都是或者 ||
percentage 百分比
length 数值
left center/right bottom

  • background-origin 9+

    属性：
padding-box(padding) 默认值 从border内边缘开始显示背景图片
border-box(border) 从border外边缘开始显示背景图片
content-box(content) 从content外边缘（padding 内边缘）开始显示图片

  • background-clip 9+

    属性：
padding-box(padding) 默认值 从padding区域向外裁剪
border-box(border) 从border区域向外裁剪
content-box(content) 从content区域向外裁剪

    • background-size 9+

      属性：
auto 自动
length 数值
percentage 百分比
cover 将图片放大 平铺整个容器
contain 保持背景图片本身宽高比例
