CSS3-文本

CSS 基础属性

font-family 字体类型 
font-style 字体样式 normal italic oblique
font-weight 字体粗细 normal bold bolder lighter 
font-size-adjust 定义强制对文本使用同一尺寸
font-stretch 是否横向拉伸
font-variant 字体大小  normal small-caps 小型的大写字母字体

CSS 文本类型

1.word-spacing 间距 normal length（可以是负数）

2.letter-spacing 字符间距 normal length（可以是负值）

3.vertical-align 文本垂直对齐方式 

- baseline 默认值
- sub 上标对齐
- super 下标对齐
- bottom 行框低端对齐
- text-bottom 行内文本低端对齐
- top 顶端对齐
- middle 居中对齐
- 百分比
- 数字
- 长度

4.text-decoration 定义文本修饰线 

- none
- underline 下划线
- overline 上划线
- line-through 删除线
- blink 闪烁线

5.text-indent 文本首行缩进

- length 
- 百分比

6.text-align 文本水平对齐方式

- left 左对齐
- center 水平居中
- right 右对齐
- justify 两端对齐

7.line-height 行高

- normal 
- length 百分比 数字

8.text-transform 文本大小写

- none 
- uppercase 大写
- lowercase 小写
- capitalize 首字母大写

9.text-shadow 文本阴影

- color 阴影颜色
- x-offset x轴位移
- y-offset y轴位移
- blur-radius 阴影模糊半径
 （兼容版本：filter：啥都问（Color="red",Direction=数值,Strength=数值））
   - Strength 阴影强度

10.text-overflow 文本溢出

- clip 不显示省略标记 只显示裁切 6+
- ellipsis 文本溢出时候 显示省略号标记 6+

11.white-space 文本之间的空白符间距

- normal 
- nowrap 空白符合并 换行符忽略
- pre 空白符 换行符号保留
- pre-wrap 空白符 换行符保留
- pre-line 空白符合并 换行符保留

12.direction 控制文本流方向

- ltr 默认值
- rtl 文本从右熬左流入
- inherit 文本流入方向由继承获得

13.word-wrap 

- normal 
- break-word 将内容在边界内换行（不截断英文单词换行）
   注意：在pre 和table 里使用是没有效果

14.word-break 文本换行 6+

- normal 
- break-all 强行截断英文单词 达到词内换行效果
- keep-all 不允许字断开

15.white-space 

- normal 
- pre 文本空白处会被浏览器扣留 以pre里面字显示
- nowrap 文本不会换行 遇到br换行
- pre-line 合并空白符序列，但是保留换行符 8+
- pre-wrap 保留空白符 但是进行正常换行 8+
- inherit 继承父元素white-space 属性值
